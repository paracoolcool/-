# myMall -------- 基于servlet + jsp + jquery的网上商城系统
该项目为以淘宝、京东等为参照的小型B/S结构应用系统，实现基本的登录注册、数据库增删查改、商城应用基本的商品展示、模拟购买以及购物车、订单、收货地址的管理功能。
# 完成功能
  数据库连接和增删查改操作
    用户注册登录
    商品列表展示
    商品详细信息
    商品下单及加购
    收货地址管理
    订单管理
  session使用
  cookie使用
  基于servlet的文件上传
  基于servlet的文件下载
    订单报表下载为excel
  web过滤器
  jQuery Ajax 使用
